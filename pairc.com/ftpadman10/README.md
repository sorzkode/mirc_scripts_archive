==INSTALLATION==



Be sure to uninstall any previous versions of Ftpadman before upgrading to 1.0.



Install like you would any other plugin for Peace and Protection.



1. Unzip the files to the /addon directory of your p&p client.

2. Start p&p.

3. Go to P&P TOOLS / SCRIPT / INSTALL / ADDONS..

4. Double click 'Ftp Ad Manager' from the list

5. That's it!



==INSTRUCTIONS==

I designed this plugin to make advertising an FTP in a channel as simple

as it possibly could.  Everything is configured through popup windows.



1. Go to P&P TOOLS / FTP AD MANAGER



You should notice a new menu item on this list called FTP AD MANAGER.



2. CONFIGURE

   This menu will take you through the channel configuration

   routine. You must input something in each window when prompted.

   If you make a mistake, just redo the whole group over again. The popups

   are pretty self explanatory, just read them carefully.



3. ON  -  This will announce to the channel that your ftp is online, and turn on a 

   timer according to the settings you previously selected.  When a user types

   the trigger they will be privately notified with the information they need.



4. OFF - This will announce to the channel that your ftp is offline, and turn off the

   timer.  Now, when a user types the trigger they will be privately notified that the

   ftp is offline.
