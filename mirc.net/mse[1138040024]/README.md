  How to use...
mIRC Script Editor (mSE) will add minimize and maximize buttons to the script edtior dialog. (khaled should have added this in the 1st place :P)

Load Dll
Syntax: //dll mSE.dll Editor .

This will load the dll, and add the buttons to the editor.

UnLoad Dll
Syntax: //dll -u mSE.dll

This will unload the dll. Eventhough mIRC will do this on edit. it's wise to type it on exit. (just incase shit).

DllInfo
Syntax: /dll mSE.dll DllInfo .

Example:
/dll mSE.dll DllInfo .
Shows info about the DLL.
