Dialog Maker V0.21 Help + Info file.



*INTRO*

The most tedious brain numbing job with scripting is making a dialog. Having to set cords

heights and widths for everything then checking it changing it etc is now in the past.

With this Dialog Maker unlike others available on Xcalibre it allows you to buils a good

looking dialog with a few clicks of the mouse.



*INSTALLATION*

The files provided within the zip/exe file must be extracted into a directory called dmaker

which should be within your mIRC directory.



To install this script type

/load -rs dmaker\install.ini

into any mIRC window to load the dialog maker. Click Yes to the confirmation prompt.
