1. Whats this ?



   This is a mIRC-Addon.



2. What does it do ?



   This Addon let's you show your current WinAMP status to a channel or query.

   This will be done in the following format:

   WinAMP Version - Artist & Title - Percent elasped of total time - Graphic indicator.



3. How does it work ?



   This Addon uses a mIRC dll, dll's are enhancements to mIRC, they add new features to

   mIRC, this Addon uses "LightStrike Swamp v1.0 DLL"



4. How to install



   Unzip the zipfile to your mIRC dir, a subfolder called "ampstatus" should be automatically

   created within your mIRC dir while unzipping.

   After the file has been unzipped fire up mIRC and type: "/load -rs ampstatus\ampstatus.mrc"



5. How to show my stuff ?



   After the script has been loaded, a "winamp" option is added to all popup menu's, from

   there you can use this Addon.

   You can also type "/sayamp" to use it.
