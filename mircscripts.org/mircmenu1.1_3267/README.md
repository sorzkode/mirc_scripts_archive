How to use:

Just start mIRCmenu.exe and open mirc scripts editor, then click CTRL+SPACE and you can see help menu. Menu will be filtered by the part of the word about position of caret, if it will be the begining of the new line or space symbol then menu will be with all commands, events, identifies. 

Also, when you call menu you may push buttons and menu will be filtered about you just have written. To choise the command, event, identify you need, just push ENTER or double click on it. 

When you will position the cursor on word in menu you will see little hint. All hints you can edit in options dialog. The syntax of this file is: <1>~<2>~<3>

      <1> - text, which you will see in menu

      <2> - text, which you will see in header of the hint

      <3> - text of the hint to command
