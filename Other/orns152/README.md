	 O.R. NickServ is a bot for mIRC that permits users to "register" a

 file 'updates.txt'.



-Setup- 



 Step 1. Open the file "orns152.zip" with WinZip or PKZip and extract it to  

         its own directory (you've probably already done this).



 Step 2. Copy the mIRC executable(mirc32.exe or mirc.exe) to the directory. 



      Note: If you don't have the mIRC executable, you can find download 

            sites at http://www.mirc.co.uk/.



 Step 3. Run the mIRC executable and connect to your server.



 Step 4. Once connected, issue the /oper command so that your bot has IRCop

        status.
