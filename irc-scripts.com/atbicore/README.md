AtbiCore is an addon designed to restrict the access to files to groups of users in the

fileserver (any one the switchs the 'fserve' event).

The script lets you add user and files to groups , when each user in a certain group - 

cannot access the files in the same group.







Through the 'AtbiCore File Manager' you can add and remove files from groups and through

several menus (nicklist,channel) you can view , add... or remove users from groups.



++File Manager.



-- To add a file to a group simply check the Add radio button , put the group's name in the 'Group' field - then click once on the file.



-- To remove a file form a group you should check the Remove radion button , enter the group name in the group field  and click once on the file.



-- To jump to a location in you hard drive just enter the location in the upper combo and press the goto button.
