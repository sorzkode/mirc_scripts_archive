Cybercow Away System v. 4 Gold

-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-



To load this script, type /load -rs <script's location and name>



On the first time you load this script, you get to choose a few things.

You do not have to have a pager sound if you don't want one.  You must,

on the other hand, set your away nick name, an important part of this

script, and you must set your message timer.  The message timer can be

set for anywhere between 1 and 9999 seconds. I personaly sugest 1500

seconds, but thats just my opinion.  Your away nick can be anything you

like.  I sugest you set it as something like the first 3 letters of the

nick you normaly use on IRC and away (Example of away nick config.

Original nick is Robert, away nick is Robaway.)  When you are setting

yourself away, you can do so from your Commands menu, by right clicking

in whatever channel you are currently in and selecting the cybercow

Away System menu, or by hitting the key F3 (if it isn't previously

configured for a different function).  To set yourself back, you can

go to either of the same menus that you set yourself away with or hit

the F4 key.  When you set yourself away, the away system will record

your current IRC nick.  When you set yourself back, it will set your

nick to the original one it recorded.  You can configure any part of

the away system in a submenu of the away system menus.  In there, you

can change everything.  There is also a pager submenu in which you

can turn off your pager.  This away system can display a message, which

you configured on loading.  By defult, your away messages are on.  If

you wish to turn them off, use the away systems options submenu and

and then to the away message options menu, then turn it off.  You turn

them on and configure the time the exact same way.  The final thing

I'll mention here is the Away Status window.  It is brought up through

the options submenu.  It pops up a window that will show you all the

option's status.
