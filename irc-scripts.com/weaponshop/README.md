|instuctions|

-------------



1.Unzip the entire folder to your mIRC folder and unzip this file into the folder aswell, for

future refrance.

2.Open mIRC

3.Click 'Tools' on the top bar and go to 'Remote...'

4.In the window that pop's up click 'file', go down to 'load' and click 'scripts'

5.Open the file WeaponShopMenu.ini

It should be in your mIRC folder under WeaponShop

6.Click OK to close the mIRC editor

7.Right Click the channel backgroun, go to 'Weapon Shop' and click ON, to turn it on.

To turn it off, go to the same menu and click OFF.

3.Type /load -rs C:\<mIRCDIR>\WeaponShop\WeaponShopMenu.ini



Where it says <mIRCDIR>, enter the address where you unziped WeaponShop, where mIRC32.exe is.



4.Right Click the channel background, go to 'Weapon Shop' and click ON, to turn it on.

To turn it off, go to the same menu and click OFF.



Either method will work
