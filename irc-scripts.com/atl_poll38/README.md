 -----====== Atl Poll v3.8 by CQAtlantis ======----

    

 Info:

 -----

    

    This is the simplest and easiest to use poll

  script ever made. To conduct a poll all you need

  to do is goto CHannel Menu > Atl Poll > Start.



 Install:

 --------

    

    To install put the file atl_poll38.mrc in your

  mirc directory. Then Type: 

    

               //load -rs atl_poll38.mrc

    

 --------------------------------------------------

    

 Use:

 ----

    

    To use this script type [ /apstart Topic ] or

  tight click in the chat window or any window and

  goto Atl Poll and click on [ Start ].



    You will then be promted to fill in a topic

  and four options. Once finished the poll will

  display the poll topic and options so that the

  users can vote. People will vote by typing "!"

  then the number by the option they want to vote

  for. The script dosn't allow anyone to vote

  twice. It stores the users host address so even

  if they change  nicks they still wont be able to

  vote again. 

    

 --------------------------------------------------
